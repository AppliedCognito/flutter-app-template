# flutter-app-template
A boilerplate Flutter project with:

- GoRouter for navigation
- Riverpod for state management
- Clean folder structure
- Light/Dark theme
- Gap for spaces

## Structure
```
lib/
├── core/
│   ├── router/
│   └── theme/
├── features/
│   └── counter/
│       ├── presentation/
│       └── providers/
└── main.dart
```
